## Overview and Usage

Dashboard cards are arranged on a dashboard. This section provides guidance on a subset of dashboard functionality, and does not currently include editable dashboard requirements.

## Description

The following are suggested visualizations of common attributes on a dashboard card:

  
</br> 

<div class="row">
  <div class="col-md-7">
    <p>
      <b>Style</b>
    </p>
    
    <ol>
      <li>
        <b>Background and Borders:</b> All cards on a dashboard should have the same background and border treatment.
      </li>
      <li>
        <b>Top Accent</b> (optional): Add a top accent to a row of cards to give them visual hierarchy
      </li></div> 
      
      <div class="col-md-5">
        <a "/wp-content/uploads/2015/07/dashCard_1.png"><img src="/wp-content/uploads/2015/07/dashCard_1.png" alt="dashCard_1" class="alignnone size-full wp-image-4087" /></a>
      </div></div> 
      
      <br /></br> <br /></br> 
      
      <p>
        <b>Titles</b>
      </p>
      
      <ol>
        <li>
          <b>Title</b> (optional): If the content of the card is not obvious, add a title to clarify what information is being presented. The title is always followed by a horizontal separator.
        </li>
        <li>
          <b>Subtitle</b> (optional): Add a subtitle if the card contains the same visualization for different object types.
        </li>
      </ol><a "/wp-content/uploads/2015/07/dashCard_titles.png">
      
      <img src="/wp-content/uploads/2015/07/dashCard_titles.png" alt="dashCard_titles" class="alignnone size-full wp-image-4134" /></a> <br /></br> <br /></br> 
      
      <p>
        <b>Timeline Legend</b> (optional):
      </p>
      
      <p>
        There are two options for positioning the timeline legend on the card:
      </p>
      
      <ol>
        <li>
          The timeline legend can be placed left-aligned under the card title.
        </li>
        <li>
          Or, it could be place under the trend chart and left aligned.
        </li>
      </ol>

      <br /></br> <br /></br> 
      
      <p>
        <b>Timeline Filter</b> (optional):
      </p>
      
      <p>
        There are two options for positioning the timeline filter on the card:
      </p>
      
      <ol>
        <li>
          If there is a card title, the filter should be displayed on the right of the title panel.
        </li>
        <li>
          If there is an action panel, the filter can be displayed on the right of the action panel.
        </li>
      </ol>

          <p>
            <b>Actions Panel</b> (optional):
          </p>
          
          <ol>
            <li>
              An actions panel can be placed at the bottom of a card.
            </li>
            <li>
              The action text may be preceded by an associated icon if applicable.
            </li>
            <li>
              The action is placed on the left side of the card and the text is displayed as a link.
            </li>
          </ol>
